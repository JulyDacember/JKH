# ЖКХ Mobile App

Мобильное приложение для центра ЖКХ для коммуникации с жителями дома.

## Основные функции

- **Авторизация пользователя** - экран входа с валидацией email и пароля
- **Главная страница** - отображение текущего баланса, активных запросов и быстрых действий
- **Просмотр баланса** - информация о текущем балансе и дате платежа
- **Отслеживание запросов** - статистика по активным заявкам (всего, в ожидании, в работе, завершено)
- **Быстрые действия**:
  - Подача заявки
  - Чтение счетчиков
  - Оплата
  - Экстренная связь
- **Показания счетчиков** - просмотр счетчиков (электричество, вода, газ) с возможностью фотографирования
- **История активности** - список последних действий и событий

## Установка и запуск

### Требования

- Flutter SDK (>=3.0.0)
- Dart SDK

### Шаги установки

1. Установите зависимости:
```bash
flutter pub get
```

2. Запустите приложение:
```bash
flutter run
```

## Особенности реализации

- **Авторизация**: Использует SharedPreferences для сохранения состояния авторизации
- **Mock данные**: Все данные представлены через MockDataService для демонстрации
- **Навигация**: Bottom navigation bar с центральной кнопкой быстрого действия
- **UI/UX**: Дизайн соответствует предоставленным макетам с современным интерфейсом

## Основные экраны

1. **Экран авторизации** - вход по email и паролю
2. **Главный экран** - дашборд с балансом, запросами и быстрыми действиями
3. **Показания счетчиков** - список счетчиков с возможностью фотографирования
4. **История активности** - список последних событий
5. **Список запросов** - все заявки пользователя
6. **Платежи** - история платежей и оплата

## Цветовая схема

- Основной цвет: `#1E3A8A` (темно-синий)
- Успех: `#10B981` (зеленый)
- Предупреждение: `#FFA500` (оранжевый)
- Ошибка: `#EF4444` (красный)
