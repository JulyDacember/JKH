# Тестовый пользователь для админ панели

## Доступ администратора

Для локальной проверки админского функционала используйте email, который начинается с `admin@` или содержит `.admin@`.

Примеры:

- `admin@zhkh.local`
- `qa.admin@zhkh.local`

Пароль в текущей демо-реализации может быть любым валидным (минимум 6 символов), например: `123456`.

## Проверка

1. Войти под admin email.
2. Открыть меню профиля в хедере.
3. Убедиться, что виден пункт `Админ панель`.
4. Перейти на `Админ панель` и проверить отсутствие сообщения о запрете доступа.

## Проверка non-admin

1. Войти с обычным email, например `user@zhkh.local`.
2. В меню профиля пункта `Админ панель` быть не должно.
3. При прямом переходе на админ-экран должно отображаться сообщение о запрете доступа.
