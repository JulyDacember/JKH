# Инструкция по установке и запуску

## Первоначальная настройка

1. Убедитесь, что у вас установлен Flutter SDK:
```bash
flutter --version
```

2. Если Flutter не установлен, следуйте инструкциям на [flutter.dev](https://flutter.dev/docs/get-started/install)

## Установка зависимостей

Выполните в корневой директории проекта:

```bash
flutter pub get
```

## Запуск приложения

### На эмуляторе/симуляторе

1. Запустите эмулятор Android или симулятор iOS
2. Выполните:
```bash
flutter run
```

### На физическом устройстве

1. Подключите устройство через USB (Android) или подключите iOS устройство
2. Убедитесь, что устройство распознано:
```bash
flutter devices
```
3. Запустите приложение:
```bash
flutter run
```

## Тестовые данные для входа

Приложение использует mock данные, поэтому для входа можно использовать любой email и пароль (не менее 6 символов).

Например:
- Email: `test@example.com`
- Пароль: `123456`  

## Структура данных

Все данные в приложении являются тестовыми (mock) и хранятся в `lib/services/mock_data_service.dart`.

## Возможные проблемы

1. **Ошибки зависимостей**: Выполните `flutter clean` и затем `flutter pub get`
2. **Проблемы с Android**: Убедитесь, что Android SDK правильно настроен
3. **Проблемы с iOS**: Требуется Mac и Xcode для разработки под iOS
